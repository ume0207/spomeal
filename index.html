⚙️ Google Cloud Console 設定手順
① console.cloud.google.com → 新規プロジェクト作成（例: スポミル）
② 「APIとサービス」→「ライブラリ」→ Google Calendar API を有効化
③ 「APIとサービス」→「OAuthの同意画面」→ 外部・アプリ名: スポミル・スコープ: calendar.events を追加・テストユーザーに spomeal20260323@gmail.com を追加

⑤ 承認済みJavaScriptオリジンに http://localhost:3341 を追加
